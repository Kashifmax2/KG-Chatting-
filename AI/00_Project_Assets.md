Brand Rules:
- Always use KG Chating as the application name.
- Never use Discord branding or Discord logos.
- Use kg-chating-logo.png as the primary logo.
- Use default-avatar.png whenever a user has no profile image.
- Use login-banner.png on authentication screens.
- Use favicon and manifest files for browser and PWA support.

Theme:
Modern, clean, premium, Discord-inspired but original UI.
